# Pipeline CI/CD — Validation HTML & Déploiement Vercel

## Présentation
Ce projet utilise un pipeline **CI/CD** complet pour garantir un code propre, testé et automatiquement déployé.  
Il repose sur deux éléments principaux :

- **CI (Continuous Integration)** : validation automatique du HTML via GitHub Actions.
- **CD (Continuous Deployment)** : déploiement automatique du site sur **Vercel**.

L’objectif est d’assurer une mise en production fiable, rapide et sans intervention manuelle.

---

Ce projet est un test pour en apprendre plus sur ces outils.
